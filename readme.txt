== Description ==
YouTu-CDN是一款WordPress腾讯万象优图图床插件。

== Installation ==
1. 通过 WordPress 插件库或者手动安装 YouTu-CDN 并启用
2. 启用后将自动跳转至设置页面，按照要求配置好接口信息
3. Enjoy.
